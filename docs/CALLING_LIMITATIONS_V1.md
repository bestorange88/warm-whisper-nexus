# Archimi Chat V1 通话功能限制说明

## 版本

V1.0 — 首发版本

## 支持的能力

- ✅ 一对一语音通话
- ✅ 一对一视频通话
- ✅ 静音/取消静音
- ✅ 开关摄像头
- ✅ 切换前后摄像头
- ✅ 接听/拒绝/挂断
- ✅ 通话消息记录

## 不支持的能力

以下功能在首发版本中**不包含**，后续版本按需评估：

| 能力 | 状态 | 说明 |
|------|------|------|
| 群语音通话 | ❌ 不支持 | 一对一优先 |
| 群视频通话 | ❌ 不支持 | 一对一优先 |
| 屏幕共享 | ❌ 不支持 | 非首发能力 |
| 通话录制 | ❌ 不支持 | 隐私与合规考虑 |
| 推流直播 | ❌ 不支持 | 非产品定位 |
| 互动礼物/打赏 | ❌ 不支持 | 非产品定位 |
| 美颜滤镜 | ❌ 不支持 | 非首发能力 |
| 通话转接 | ❌ 不支持 | 非首发能力 |
| 通话推送通知 | ❌ 不支持 | 架构已预留，待 V2 |
| 后台保活 | ❌ 不支持 | iOS 限制，需 CallKit |
| 通话中发消息 | ❌ 不支持 | 非首发能力 |
| 多人会议 | ❌ 不支持 | 非产品定位 |
| 聊天室 | ❌ 不支持 | 非产品定位 |
| 随机视频匹配 | ❌ 不支持 | 非产品定位 |

## 已知限制

1. **通话推送**：当前版本 App 在后台时无法收到来电通知，需要 App 在前台运行
2. **网络切换**：WiFi/4G 切换时可能导致通话短暂中断
3. **并发通话**：同一用户同时只能进行一个通话
4. **浏览器兼容性**：WebRTC 在部分旧浏览器中不支持

## 后续规划

- V1.1：通话推送通知（FCM/APNs）
- V1.2：通话质量监控与统计
- V2.0：群语音/视频通话（评估中）
